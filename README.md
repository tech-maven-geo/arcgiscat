# arcgiscat
This is a demo app to showcase loading JSON Catalog Layer
create a vuejs esri arcgis maps sdk mapping app that can demonstrate loading catalog layer (JSON)
add catalog from local or from remote hosted URL and show a table of contents of the data layers
for users to manage the data (toggle on/off, control opacity, control layer order, zoom to, etc)
https://developers.arcgis.com/javascript/latest/sample-code/layers-cataloglayer/
https://developers.arcgis.com/javascript/latest/api-reference/esri-layers-CatalogLayer.html 
please create several sample JSON catalog for testing that can show both 
ESRI and OGC mapping services, tile layers, vector tiles, static files (geojson, kml, csv, etc)
and 3DTILES and SceneServer and real-time data StreamServer.
